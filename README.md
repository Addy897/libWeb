# libWeb
Cross Platform Micro web framework written in C



# Introduction 

libWeb is MicroWeb framework written in C to serve http based web apps.
It can be used with any frontend library like React. 






# compile
## libWeb
  ```bash
  make
  ```
## example
  ```bash
  make example
  ```
# Run Example
 ```bash
 cd example
 ./main
 ```
